# Java Programs Repository

This repository contains a collection of Java programs created for learning and practicing Java programming concepts.

The programs included here cover:

- Basic Java Programs
- Conditional Statements
- Loops
- String Handling
- Array Programs


## Repository Purpose

The main purpose of this repository is to:

- Practice Java programming
- Improve problem-solving skills
- Understand core Java concepts
- Build a strong programming foundation


## Technologies Used

- Java
- JDK
- VS Code / IntelliJ IDEA / Eclipse
