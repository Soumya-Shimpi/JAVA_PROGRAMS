# InfyTV - Encapsulation and Abstraction in Java

## 📌 Project Overview

This project demonstrates the concepts of **Encapsulation** and **Abstraction** in Java using an `InfyTV` class. The program stores information about a documentary film team, including the correspondent, news reporter, and photographer, and displays the documentary details.

## 📂 Project Structure

```
EncapsulationAndAbstraction/
│
├── InfyTV.java
└── Tester.java
```

## 🚀 Features

* Uses **private data members** to achieve encapsulation.
* Provides **getter and setter methods** to access and modify data.
* Demonstrates method invocation through an object.
* Displays documentary information along with team member details.

## 🛠 Technologies Used

* Java
* Object-Oriented Programming (OOP)

## 📋 Class Description

### InfyTV Class

Contains:

* `photographer`
* `newsreporter`
* `correspondent`

Methods:

* Getter methods for all attributes
* Setter methods for all attributes
* `documentaryFilm()` method to display documentary information

### Tester Class

* Creates an object of `InfyTV`
* Sets correspondent, reporter, and photographer names
* Calls `documentaryFilm()` to display the output
